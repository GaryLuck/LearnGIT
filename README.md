# LearnGIT

A personal sandbox repository for learning Git and experimenting with small programs. This is not production code — just a place to practice version control and try things out.

## Repository Structure

```
experiments/   Small, self-contained programs and visualizations
  hello.c          "Hello World" in C
  count.c          Count from 1 to 10 in C
  mandelbrot.py    Interactive Mandelbrot set viewer (matplotlib)
  sierpinski.py    Sierpinski triangle generator (matplotlib)
```

Future additions may use:
- `notes/` — text or markdown learning notes
- `scripts/` — utility shell/Python scripts
